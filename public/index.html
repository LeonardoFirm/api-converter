Descubra o significado das extensões mais usadas: JPG, PNG, WEBP e ICO

JPG

Joint Photographic Experts Group

A extensão JPG foi atribuída para ficheiros de imagens. Muitas fotos e gráficos web são guardados em JPG. A fim de comprimir muitos bitmaps são guardados em .jpg, o que torna mais fácil transferir e descarregar estes ficheiros na Internet. Formato JPG baseado na paleta de cores de 24-bits, quanto mais alto o nível de compressão aplicado para criar o ficheiro JPG, maior o efeito de descompressão sobre a qualidade da imagem.

PNG

Portable Network Graphic

PNG é um formato de armazenamento de dados de gráficos raster que utiliza o algoritmo de compressão sem perdas para Deflacionar. O PNG foi criado como um formato livre para substituir o GIF. O PNG suporta três tipos principais de imagens raster: imagem a preto e branco, uma cor de imagem indexada e imagem de cores. Formato PNG armazena informações gráficas numa forma comprimida.

WEBP

Formato de Imagem WebP

Este é um novo formato que suporta compressão de qualidade sem perdas e com perdas para imagens na Internet. A empresa Google desenvolveu este formato especificamente para funcionar on-line tão rápida e convenientemente quanto possível. A principal vantagem é que o seu tamanho de ficheiro é pequeno comparado com outros formatos de imagem, mas semelhante na qualidade da imagem.

ICO

Ícone

ICO é um ficheiro deste formato que contém normalmente uns pequenos ícones de imagem de resoluções diferentes (16x16, 32x32, 64x64 pixels) e diversas profundidades de cor (16 cores, 32, 64, 128, 256, 16-bit, e assim por diante). Usado para exibir ficheiros e pastas nos sistemas operativos para interface de utilizador gráfica (GUI). Este formato pode ser aberto com a ajuda do programa ACDSee.
